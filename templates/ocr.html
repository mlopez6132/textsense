Technology

Image-to-text extraction is powered by PaddleOCR running on the server. It performs text detection and recognition with angle classification for robust results.

  • Pipeline: angle classification → text detection → text recognition
  • Optimized for CPU inference; starts quickly and is lightweight
  • Outputs are combined in natural reading order
