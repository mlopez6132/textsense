How to Use
Input Methods
  • Direct Text: Paste or type text directly into the analyzer
  • File Upload: Upload .txt, .md, .doc, or .docx files

Understanding Results
  • Color Coding: Green = Human-like, Red = AI-like
  • Percentages: Show probability for each sentence
  • Overall Score: Average AI probability across all text
Best Practices
  • Analyze complete sentences or paragraphs
  • Consider context and content type
  • Use results as guidance, not absolute truth
  • Review highlighted sections carefully
Export Options
  • Copy analyzed text to clipboard

  • Keep detailed analysis for records
Technology

TextSense uses the Desklib AI Text Detector fine-tuned on microsoft/deberta-v3-large. The model aggregates token embeddings with mean pooling and applies a classifier head to output an AI probability.

  • Transformer base: DeBERTa v3 Large
  • Mean pooling over attention-masked hidden states
  • Sentence-level probability via sigmoid on classifier logit
